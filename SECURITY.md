# Security Policy

This repository is an index and does not ship executable integration code.

However, listed projects may execute commands, access accounts, read or write data, transfer assets, or sign transactions. Treat all listed tools as potentially sensitive.

## Reporting Security Issues

If you find a security issue in this repository's data, such as a malicious link, impersonating repository, or unsafe classification, please open an issue with:

- The affected entry.
- The source URL.
- The risk.
- Suggested correction.

For vulnerabilities in listed third-party projects, report them to the upstream project or vendor.

## High-Risk Categories

Use extra caution with:

- Crypto wallets, exchanges, transfers, or on-chain signing.
- Financial trading and order placement.
- Payment systems.
- Production cloud accounts.
- Business documents, customer data, employee data, and private repositories.
