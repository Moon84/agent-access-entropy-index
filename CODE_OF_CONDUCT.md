# Code of Conduct

We aim to keep this project factual, careful, and useful.

## Expected Behavior

- Be respectful.
- Provide evidence for corrections.
- Distinguish official, unconfirmed, and community sources.
- Avoid promotional language.
- Avoid unsafe claims such as "secure", "audited", or "recommended" unless backed by evidence.

## Unacceptable Behavior

- Harassment or abuse.
- Vendor impersonation.
- Submitting malicious links.
- Misrepresenting community projects as official.
- Encouraging unsafe use of trading, payment, wallet, or production-write tools.
