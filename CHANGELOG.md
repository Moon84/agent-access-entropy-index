# Changelog

All notable changes to this project are documented here.

This project uses simple semantic versioning for the dataset and repository package.

## 0.1.0 - 2026-07-07

- Initial local draft of Agent Access Entropy Index.
- Added primary unified dataset: `data/01-access-resources.csv`.
- Added derived machine-readable exports: JSONL, SQLite, and JSON Schema.
- Added local stdio MCP server for agent-friendly querying.
- Added README, Chinese README, methodology, verification policy, disclaimer, security policy, and issue templates.
