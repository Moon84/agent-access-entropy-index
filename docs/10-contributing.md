# Contributing

Thank you for helping improve Agent Access Entropy Index.

## What To Submit

Good contributions include:

- New official CLI, Skill, MCP, SDK, or API sources.
- Corrections to official status.
- Updated links.
- Risk notes for write-capable, trading, payment, wallet, or sensitive-data tools.
- Evidence that a listed source is not official.

## Evidence Requirements

Please include at least one evidence source:

- Official GitHub organization.
- Official vendor domain or documentation.
- README statement from the product team.
- Release notes or install documentation from the vendor.

## Suggested Fields

For CSV updates, include:

- `domain` or `industry`
- `platform` or `company`
- `official_cli`
- `official_skill`
- `official_mcp`
- `official_source_url`
- `verification_status`
- `notes`
- `checked_at`

## Risk Labels

Please call out high-risk capabilities:

- Writes to production data.
- Executes shell commands.
- Accesses cloud accounts.
- Sends messages or emails.
- Handles customer or employee data.
- Places orders, trades assets, transfers funds, or signs blockchain transactions.
