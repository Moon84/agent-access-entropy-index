# Verification Policy

This project uses conservative labels.

## Confirmed Official

Use "confirmed official" only when at least one condition is met:

- The repository is under the vendor's official GitHub organization.
- The source is linked from the vendor's official website or documentation.
- The README clearly identifies the repository as maintained by the official product team.

## Partially Confirmed

Use "partially confirmed" when:

- The API or SDK is official, but the Skill or MCP wrapper is not confirmed.
- The platform has official documentation but the GitHub repository belongs to a related team or sub-brand.
- The source is hosted on an official domain but not mirrored on GitHub.

## Unconfirmed

Use "unconfirmed" when:

- The repository is third-party.
- Official status is claimed but not evidenced.
- The source wraps an official API but is not maintained by the vendor.

## Community

Use "community" when:

- The project is useful but clearly third-party.
- No official Skill, CLI, or MCP has been found.
